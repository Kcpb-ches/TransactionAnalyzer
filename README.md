# 📊 洗錢防制與交易明細分析系統 (Transaction Analyzer)

這是一個專為執法單位、金融防制 (AML) 與反詐欺調查設計的**「全前端、純離線」**金流視覺化分析工具。

透過現代化網頁與 ECharts 技術，調查人員只需將調閱到的銀行交易明細與開戶資料拖曳至網頁中，系統便會利用瀏覽器內建資料庫 (IndexedDB) 即時聚合龐大的交易資料，並繪製出直觀的金流圖表。

> **⚠️ 資安保證**：本系統為 100% Client-side (前端) 運算。所有的 CSV 解析、資料清洗、儲存與圖表渲染都只在您的個人電腦與瀏覽器記憶體中進行，完全不會將機密的金融資料上傳至任何雲端伺服器，最高規格保障案件機密。

## ✨ 核心功能特色

*   **📂 智慧檔案解析與本地資料庫**
    *   支援上傳 `*.csv` 與 Excel `*.xlsx` 格式檔案。
    *   自動判定「開戶資料」、「交易明細」或「自訂帳戶字典」進行分類清洗。
    *   內建台灣主要銀行代碼對照庫，對於未記載銀行代碼的清單能自動透過帳號前綴進行智慧反查與中文名稱補齊。
    *   **資料保護與合併**：支援不同批次來源的檔案匯入，且能智慧合併開戶清單，保留字典中的銀行代碼而不被覆蓋。

*   **📈 單一帳戶深度儀表板分析**
    *   以動態長條圖呈現：**「存入頻率 Top 5」**、**「轉出頻率 Top 5」**、**「存入總額 Top 5」**、**「轉出總額 Top 5」**。
    *   **防制洗錢特徵統計 (AML)**：自動鎖定交易摘要中的關鍵字 (如：現金、ATM、提現等)，並獨立統計異常現金進出次數與總額。
    *   支援將「結清」狀態之帳戶自分析選單中自動過濾剔除。

*   **🌊 資金水房流向分析：桑基圖 (Sankey Diagram)**
    *   針對所選擇的主帳戶，將龐大交易明細化繁為簡，轉化為完美對稱的金流流向圖。
    *   **左側 (匯入來源) ➔ 中央 (主帳戶過水樞紐) ➔ 右側 (匯出/提領去向)**，連線粗細直觀對應總金額大小。
    *   **智慧收斂機制**：自動抓出前 15 大匯出行與匯入行獨立顯示，並將數百筆零散小額匯款、轉帳手續費自動彙整為灰色的「其他零散或無名交易」節點，確保版面乾淨並凸顯大額不明金流。

*   **🕸️ 跨帳戶全域資金網絡分析 (Global Network Graph)**
    *   **專業級圖論網路渲染 (Cytoscape.js)**：摒棄一般商業圖表的限制，全面採用生物資訊領域與專業資安風控首選的 **Cytoscape.js** 建構金流共犯結構網。
    *   **高階向心同心圓佈局 (Concentric Layout)**：自動根據每個帳戶的連結頻率 (Degree) 計算權重，將資金來往最頻繁的「超級樞紐 (水房/大車手)」強制置於陣列中心，而單一無關聯的「葉節點 (Leaf Nodes)」則如行星般散布在最外圍同心軌道，犯罪核心一眼看穿。
    *   **完美靜態拖引 (Pin Nodes)**：支援完美的滑鼠拖曳與縮放操作，所有節點只要被您拖拉展開後就會像釘死在畫布上一樣，**徹底消除傳統力導向圖 (如 ECharts) 無限排斥回彈、牽一髮動全身的橡皮筋拉扯感**，讓您能專心將特定嫌疑人「排排站」進行分類。
    *   **動態關聯高亮 (1st-Degree Hover Highlight)**：當游標懸停在任何節點或連線上時，系統會瞬間高亮該節點與「所有一層直接交易對象」，並同時使周圍毫無關聯的其他集團成員與金流隱身淡化 (Opacity Fade-out)，提供極度專注的解析視野。
    *   **智能降噪過濾器**：圖表上方附帶動態參數控制器，可隨時拉高「單線金額下限」或「最少交易次數」，過濾掉幾十元的手續費雜訊或過客帳戶，避免瀏覽器在面對幾萬筆資料時出現效能雪崩。

> **💡 開發日誌與技術選型：為何放棄 ECharts 放抱 Cytoscape？**
> 在早期的版本中，網脈圖曾採用 Apache ECharts 的力導向圖 (`force-directed`)。然而我們發現，當執法單位匯入龐大的真實洗錢網路時，ECharts 為了維持畫面的「動態物理平衡」，會導致嚴重的節點拉扯，使得使用者無法將特定的子網群組（例如 A 集團與 B 集團）獨立拖曳分開；且 ECharts 在處理數千條雙向箭頭與標籤時會出現畫面過載現象。因此，我們在後續的更新中全面改寫底層，引入了專注於 Graph Theory (圖論) 的 **Cytoscape.js**，不只達成了「算完即靜止」的零回彈排版，更加入了強大的向心階層排列與游標高亮，帶來了真正業務級的巨量金流分析體驗。

## 🚀 如何使用本系統

本專案支援免安裝即開即用，您可直接利用 GitHub Pages 公開版操作：
👉 **🔗 [點擊此處開啟線上分析系統](https://kcpb-ches.github.io/TransactionAnalyzer/)**

### 💡 自訂帳戶字典檔準備指南
如果您有自己搜集或整理的嫌疑人/車手帳戶清單，想讓系統自動幫明細圖表「翻譯」出人名與銀行，請準備一份 Excel (`.xlsx`) 或 CSV 檔案，**第一排標題請務必包含以下名稱**：
- **必備欄位**：`帳號`、`戶名` 
- **選填欄位**：`銀行代碼` (或 `銀行名稱`、`行庫代碼`)
> *提示：不論帳號前面有沒有 `0`，或是不是被 Excel 轉成公式 `="008"`，系統在匯入時都會幫您智慧清洗並補齊 16 碼！*

### 📌 操作步驟
1. 將您手邊整理好的「開戶資料(人頭庫)」、「自訂字典」及「交易明細」CSV / Excel 檔一次拖曳到上傳虛線框內。
2. 點擊 **「確認上傳並分析」**。
3. 在下拉選單中選擇想分析的主力嫌疑人/帳戶。
4. 點擊 **「開始分析此帳戶關聯」** 即可自動展開所有儀表板與桑基圖。
